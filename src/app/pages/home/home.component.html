Bem-vindo ao Contact Manager

Sua solução completa para gerenciar contatos de forma eficiente

O que você pode fazer?

  • Adicionar contatos: Cadastre novos contatos de forma rápida e fácil.
  • Editar contatos: Atualize informações de contatos existentes.
  • Excluir contatos: Remova contatos que não são mais necessários.
  • Filtrar contatos: Encontre contatos pesquisando pelo nome ou email.
  • Exportar: Exporte contatos no formato .CSV.

Como começar?

  1. Navegue até a página de Contatos para ver sua lista de contatos.
  2. Use o botão "Criar Contato" para adicionar um novo contato.
  3. Para exportar, use o botão "Exportar para CSV".
  4. Clique em um contato para editar ou excluir.

Dicas rápidas

  • Use a barra de pesquisa para encontrar contatos rapidamente.
  • Exporte seus contatos em formato CSV.
