Bem-vindo ao Contact Manager

Sua solução completa para gerenciar contatos de forma eficiente

O que você pode fazer?

  • Adicionar contatos: Cadastre novos contatos de forma rápida e fácil.
  • Editar contatos: Atualize informações de contatos existentes.
  • Excluir contatos: Remova contatos que não são mais necessários.
  • Filtrar contatos: Encontre contatos específicos usando filtros avançados.
  • Importar/Exportar: Importe contatos de um arquivo do Google ou exporte para Excel.
  • Enviar e-mails: Envie mensagens diretamente para seus contatos.

Como começar?

  1. Navegue até a página de Contatos para ver sua lista de contatos.
  2. Use o botão "Criar Contato" para adicionar um novo contato.
  3. Para importar contatos, clique em "Importar" e selecione o arquivo do Google.
  4. Para exportar, use o botão "Exportar para Excel".
  5. Clique em um contato para editar, excluir ou enviar um e-mail.

Dicas rápidas

  • Use a barra de pesquisa para encontrar contatos rapidamente.
  • Mantenha seus contatos organizados com etiquetas e categorias.
  • Exporte seus contatos regularmente para backup.
