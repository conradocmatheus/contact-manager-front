Sobre esta página

Esta aplicação foi desenvolvida para gerenciar contatos de forma eficiente e intuitiva. No front-end, utilizei Angular com TypeScript e Bulma para criar uma interface responsiva e moderna. No back-end, a aplicação é alimentada por Node.js, Express, Prisma e SQLServer no banco de dados, garantindo uma arquitetura robusta e escalável.

Com esta ferramenta, é possível adicionar, editar, filtrar e exportar contatos. Tudo isso com uma experiência de usuário simples e eficaz.
